Python Data Types Methods Cheat Sheet

str Methods

Data Type Method Description Problem Statement Code Snippet
str strip() Removes leading and trailing whitespace characters Remove extra whitespace from user input 
user_input = input("Enter your name: ").strip()
str split() Splits a string into a list based on a separator Split a sentence into individual words 
sentence = "This is a sample sentence."
words = sentence.split()
str join() Joins a list of strings with a separator string Concatenate words into a sentence 
words = ["This", "is", "a", "sentence."]
sentence = " ".join(words)
str upper() Converts all characters to uppercase Convert a string to uppercase 
text = "hello"
upper_text = text.upper()
str lower() Converts all characters to lowercase Convert a string to lowercase 
text = "HELLO"
lower_text = text.lower()
str find() Returns the lowest index of the substring Find the position of a substring 
text = "hello world"
position = text.find("world")
str replace() Replaces occurrences of a substring Replace a substring with another 
text = "hello world"
new_text = text.replace("world", "there")

int Methods

Data Type Method Description Problem Statement Code Snippet
int bit_length() Returns the number of bits required to represent the integer in binary Find the bit length of an integer 
num = 10
bits = num.bit_length()
int to_bytes() Returns an array of bytes representing the integer Convert an integer to a byte array 
num = 1024
byte_array = num.to_bytes(2, byteorder='big')
int from_bytes() Returns an integer represented by an array of bytes Convert a byte array to an integer 
byte_array = b'\\x04\\x00'
num = int.from_bytes(byte_array, byteorder='big')

float Methods

Data Type Method Description Problem Statement Code Snippet
float is_integer() Returns True if the float is an integer Check if a float is an integer 
num = 10.0
result = num.is_integer()
float as_integer_ratio() Returns a pair of integers representing the float Get the integer ratio of a float 
num = 0.75
ratio = num.as_integer_ratio()
float hex() Returns the hexadecimal string representation of the float Get the hexadecimal representation of a float 
num = 3.14
hex_str = num.hex()

my_dict = {"a": 1, "b": 2}
items = my_dict.items()
